Тестування працездатності системи

POST: додати користувача

GET: отримати всіх користувачів

GET: отримати користувача по id

PUT: змінити користувача

DEL: видалити користувача

POST: створити коментар

GET: отримати всі коментарі

GET: отримати коментар по id

GET: отримати коментарі пов'язані з користувачем

PUT: оновити коментар

DEL: видалити коментар
